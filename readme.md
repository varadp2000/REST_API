REST API application with Express with endpoints for CREATE,SEARCH and DETELE

###'Steps to run'
'''
Clone Repository
'''


###Install node modules
'''
npm install
'''


###Start App
'''
node server.js
'''


###On postman open 127.0.0.1:8081
Screenshots are in /demo

'''
-expresss
